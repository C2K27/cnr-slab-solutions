# CNR Slab Solutions
<img width="1774" height="887" alt="b4608719-aea2-44dc-a91e-ff92ac7d38bf" src="https://github.com/user-attachments/assets/810fbef3-64b8-48d5-9e9b-6a3b99d5813c" />

Small frontend concept for a concrete slab visualization platform.

Users can preview before/after concrete installs, switch materials, browse project examples, and get fake instant pricing.

Built mostly for fun, UI practice, and GitHub portfolio stuff.

## Stack

* HTML
* CSS
* JavaScript

## Features

* Before/after slab preview
* Material selector
* Responsive layout
* Project gallery
* Dark industrial UI
* Smooth animations

## Notes

Frontend only. No backend or real ordering system connected.
